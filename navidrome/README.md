# Navidrome

[Navidrome](https://github.com/navidrome/navidrome) is a music streaming server. It is compatible with various AirSonic-compatible clients. On Android, I personally like [Subtracks](https://f-droid.org/en/packages/com.subtracks/), available from [F-Droid](https://f-droid.org).

This application is intended to be run behind an NGINX reverse proxy, provided by the [SWAG image](https://github.com/linuxserver/docker-swag).

## Install

Execute `docker-compose up -d`.
